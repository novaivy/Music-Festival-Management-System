# Music Festival Management System

Local dev: http://localhost:8080/music_festival_system/

DB: import database/music_festival_db.sql into phpMyAdmin.

Files:
- config.php : DB connection
- register.php, login.php, participant_dashboard.php, admin_dashboard.php

- assets/css/style.css
- # Music Festival Management System

## Description
A web-based system developed using PHP and MySQL to manage music festivals for schools and tertiary institutions.

## Features
- Admin management
- Judge registration
- Class management
- School participation
- Results and scoring

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP

## Installation
1. Clone the repository
2. Import database from /database/music_festival.sql
3. Run Apache and MySQL using XAMPP
4. Access via http://localhost:8080/music_festival_system/public
